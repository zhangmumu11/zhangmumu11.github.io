Linux 指令学习

2020-12-12 评论

##Linux 指令学习

mkdir 创建文件夹

mkdir 加 参数-p 可以创建多层目录

rm -rf rm 删除 -r 将目录及以下文档这逐一删除 -

-f 即文件即使为只读文件，也直接删除，无需提示

touch 创建文件

touch aaa.txt

###拷贝命令

cp 源文件夹 目标文件夹 将源文件夹中的文件拷贝到目标文件夹

cp -r 源文件夹 目标文件夹 将源文件夹中中所有的文件拷贝到目标文件夹中

剪切

mv 件夹 目标文件夹 将源文件夹中的文件剪切到目标文件夹

mv -r 源文件夹 目标文件夹 将源文件夹中中所有的文件剪切到目标文件夹中，如果是同一级目录即为重命名文件

###ls -al 展示当前文件夹的所有文件及文件夹的所有属性

d rwx r-x r-x

r读 w写 x 执行 -没有当前权限

展示第一列 【d】 为目录 【-】 为文件 【|】 为连接文档即为快捷方式

第2到4 为属主权限 rwx 1读 2写 3 执行

第5到7 属组权限 r-x 1读 2写 3 执行

第8到10 其他人权限 r-x 1读 2写 3 执行

###chgrp 命令

chgrp [][]【选项参数】【所属群组】【文件或者目录】

chgrp [][]【选项参数】【–reference==《参考文件或者目录》】【–version】【文件或者目录】

将文件夹aaa 的所属权限更改为root 权限

chgrp root aaa

将文件夹aaa 的所属权限在重新更改为admin权限

chgrp -v root aaa -v 为显示所操纵的说明

###chown

chown 属主名 文件名 更改属主

chown 【参数选项】 属主名：属组名 文件名 更改属主和属组

-R 处理指定目录及其子目录下的所有文件

数字权限

chmod -R 770 aaa 即为将aaa文件夹当前用户为 可读 可写 可执行权限 当前组为为 可读 可写 可执行权限，其他用户为不可读 不可写 不可执行 权限

符号权限

chmod -R u=rwx,g=rx,o=r bbb 即为将bbb文件夹所有者为 可读 可写 可执行 权限 当前组为为 可读 可写权限，其他用户为可读权限

chmod -R a=rwx 全部用户为可读 可写 可执行 权限 即所有用户都有权限

chmod -R o-rwx 去掉其他用户的可读 可写 可执行 权限

touch

###查看文件命令

cat -n 查看小型文件 如果查看文件 则内容展示不全 【-n】 为添加行号

less 查看大型文件 会查看全部文件内容 增加 【-N】 添加行号 上下箭头键 可以查看全部

退出查看 按q

tail 指令 不添加任何指数则显示最后10行 -3 则为显示最后3行

tail -f 则动态显示最后10行 -3 则动态显示最后3行

tail -n 则显示显示错误 -n+2 显示第二行到最后一行内容

tail -c 显示显示最后的字符 -c 45 显示最后的45个字符

head 和tail 命令相反

###grep 关键字

grep 根据关键字查看文件中的内容

grep 关键字 文件名称

grep -n 关键字 文件名称 则显示行号

grep -i 关键字 文件名称 忽略大小写

grep -v 关键字 文件名称 显示不含关键字的内容

ps -ef | grep tomcat 查看进程中含有tomcat 信息的进程

ps -ef | grep tomcat |grep -v ‘grep‘ 展示不包含grep进程的信息 即排除最后一行

ps -ef | grep -c tomcat 展示包含tomcat进程的个数

vim定位行

vim 文件名称 +5 则打开文件时光标定位到第5行

vim 异常处理

vim编辑器异常退出时则会产生swp格式的交换文件

如果插线swp格式的交换文件 则删除即可

echo命令

echo 字符串 展示字符串

ccho 字符串> 文件名称 将字符串写道文件中并覆盖文件原内容

ccho 字符串>>文件名称 将字符串写道文件中不覆盖文件原内容

cat 不存在的目录 &>>error.log 将失败的信息追加到error.log文件中

ln -s 目标文件名称 快捷方式名称 创建文件软连接

find指令

find . -name ‘a.txt’ 查找当前目录文件夹下a.txt 文件夹

##压缩文件指令

gzip a.txt 压缩a.txt文件

gzip * 压缩当前文件夹所有的文件

gzip -dv * 解压文件并列出所有信息

gunzip * 解压文件

tar

tar -cvf 打包文件名 文件名 打包文件并指定打包之后的文件名 （仅打包不压缩）

tar -zcvf 压缩文件名 文件名/文件夹名称 压缩文件或文件夹并指定压缩文件名（打包压缩）、

tar -ztvf 压缩文件名 查看压缩文件中有那些文件

tar -zxvf 压缩文件名 解压文件

参数选项 ：

-c 建立新的文件

-v 显示指令压缩的过程

-f 指定压缩文件

-z 通过gzip指令压缩文件

-t 列出压缩文件的内容

-x 表示解压

zip

zip -q -r 压缩文件名 文件/文件夹 压缩

unzip -l aaa.zip 查看压缩文件中所包含的目录

unzip -d bbb aaa.zip 解压aaa.zip到bbb文件夹中

-l 显示压缩文件夹中所包含的文件

-d <目录> 指定文件解压后所要存储的目录

bzip2 a.txt 压缩文件 bz2 格式的文件

bunzip2 -v a.bz2 解压并显示解压信息

网络

ifconfig

ifconfig 显示ip及网卡信息

ifconfig ens37 down 关闭网卡

ifconfig ens37 up 开启网卡

ifconfig ens37 up 192.168.23.99 配置IP地址

ifconfig ens37 up 192.168.23.99 255.255.225.0 配置ip和子网掩码

##ping

ping 网址 测试是否能和目标联通

netstat

netstat -a 显示详细的连接状况

netstat -i 显示网卡列表

硬盘管理

lsblk 查看硬盘

df 显示整个硬盘显示情况

df 文件夹 显示文件夹使用情况

df –total 显示所有的信息

df -h 将结果变成kb ，mb ，gb 形式显示 利于阅读

yum

yun -y install tree 安装软件tree

tree 显示目录结构

yun remove tree 卸载tree

yum list tom* 找出tom开头的软件
