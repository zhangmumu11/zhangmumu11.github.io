shell 学习

##shell定义变量

###普通变量

方式一 ： 

变量名=变量值             变量值必须是一个整体 中间没有特殊字符   abc，或者数字

方式二 ：

​ 变量名=’变量值’ 单引号中内容，原样赋值 不常用

方式三 ：

变量名="变量值"          如果双引号中有其他变量，会把变量中的结果进行拼接，然后赋值。

​ 数字不加引号 ，其他默认加双引号

命令变量

方式一：

变量名`命令` 重音符

方式二：

变量名：$(命令)

执行流程 ： 1执行重音符` 或者$() 中的 命令 2 将命令中的结果赋值给变量

例子 d= “现在的时间为$(date)

只读变量

readonly 变量名

删除变量

unset 变量名

shell 数组

定义数组

数组名=（值1 值2 值3） arr= (1 2 3 4 5)

给数组元素赋值

数组名【索引】=值 arr[0] = 3

运算符

= 检测两个字符串是否相等 ， 相等返回true [ $a = $b ] 返回结果为0和1 0为true 1为false

!= 检测两个字符串是否不相等 ，不相等返回true [ $a = $b ] 0为true 1为false

-z 检测字符串长度是否为0， 为0 返回true [ -z $a ] 0为true 1为false

-n 检测字符串长度是否不为0， 不为0 返回true [ -n $a ] 0为true 1为false

$ 检测字符串长度是否为空， 不为空 返回true [ $a ] 0为true 1为false

$? 为上一条语句的结果

[ -n $a ]

$?

关系运算符

关系运算符只支持数字，不支持字符串，除非字符串也是数字

-eq 检测两个数是否相等，相等返回ture

-ne 检测两个数是否不相等，不相等返回ture

-gt 检测左边的数是或否大于右边的数，大于返回ture

-lt 检测左边的数是或否小于右边的数，小于返回ture

-ge 检测左边的数是或否大于等于右边的数，大于等于返回ture

-le 检测左边的数是或否小于等于右边的数，小于等于返回ture

boolean运算

！ 取反运算 [ ! false ]

-o or 或运算 [ $a -lt 20 -o $b -gt 100 ]

-a and 与运算 [ $a - lt 20 -a $b -gt 100 ]

逻辑运算符

&& and [[ true && true ]] 双中括号

|| or [[ true || false]] 双中括号

shell 函数

根据索引 确定参数位置

方式一： 无参无返回值

函数名 （）{

}

例子：

abcd （）{

echo “函数”

}

调用函数

abcd 函数名称即可

方式二： 有参无返回值

abcd（）{

echo “$1”

echo “$2 ”

}

abcd a b

方式二： 有参有返回值

abcd（）{

echo “$1”

echo “$2”

return$(($1+$2))

}

abcd 10 20
